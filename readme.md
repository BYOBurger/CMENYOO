# Menu Functions and How to Use Them

## Opening and Closing the Menu

Press:

```text
K
```

to open or close the menu.

The menu can also be moved by dragging the top bar.

---

## Unload Button

The red `Unload` button removes the menu and turns off active features.

Unload removes:

```text
Player chams
Object chams
Noclip
Fly
The menu UI
The mouse unlock layer
```

---

## Visuals Tab

The `Visuals` tab controls player highlights.

### Chams

`Chams` highlights other players through walls.

Player colors are based on their Roblox team color.

Examples:

```text
Red team = red highlight
Blue team = blue highlight
No team = white highlight
```

How to use:

```text
Left-click Chams = toggle on/off
Right-click Chams = set a keybind
```

---

## Objects Tab

The `Objects` tab lets you search for objects in the map and highlight them.

### Object Search Bar

Type in the search bar to find objects by:

```text
Name
Class
```

Examples:

```text
Part
Door
Generator
MeshPart
Model
```

### Object Chams

`Object Chams` controls whether selected objects are highlighted.

How to use:

```text
Left-click Object Chams = toggle object highlights on/off
Right-click Object Chams = set a keybind
```

### Selecting Objects

Search results show objects with `[ON]` or `[OFF]`.

```text
[ON]  = highlighted
[OFF] = not highlighted
```

How to use:

```text
Left-click an object = toggle that object highlight
```

---

## Object Groups

If a bunch of objects have the same name and class, the menu groups them together.

Example:

```text
▶ [OFF] Part [Part] x12 (0/12)
```

That means there are 12 matching objects in that group.

### Group Controls

```text
Left-click group = open/close dropdown
Right-click group = toggle the whole group
```

### Group States

```text
OFF = no objects in the group are highlighted
ON  = every object in the group is highlighted
MIX = some objects are highlighted, some are not
```

### Dropdown Objects

When a group is opened, you can toggle single objects inside it.

```text
Left-click object inside group = toggle only that object
```

---

## Movement Tab

The `Movement` tab controls noclip, fly, and fly speed.

---

## Noclip

`Noclip` lets your character move through walls and parts.

Default key:

```text
H
```

How to use:

```text
Left-click Noclip = toggle on/off
Right-click Noclip = set a new keybind
Press H = toggle noclip
```

---

## Fly

`Fly` lets your character fly around.

Default key:

```text
J
```

How to use:

```text
Left-click Fly = toggle on/off
Right-click Fly = set a new keybind
Press J = toggle fly
```

### Fly Controls

```text
W = forward
A = left
S = backward
D = right
Space = up
LeftControl = down
```

---

## Fly Speed Slider

The `Fly Speed` slider changes how fast you fly.

Default speed:

```text
55
```

Slider range:

```text
10 to 200
```

How to use:

```text
Drag slider left = slower fly speed
Drag slider right = faster fly speed
```

The speed updates instantly while flying.

---

## Teleport Tab

The `Teleport` tab lets you teleport near another player.

### Player Dropdown

Click:

```text
Select Player
```

to open the player list.

The player names are colored by team.

Examples:

```text
Red team player = red name
Blue team player = blue name
No team player = white name
```

### Teleporting to a Player

How to use:

```text
Open the dropdown
Click a player name
You teleport near that player
```

The teleport places you slightly near/behind the selected player, not directly inside them.

---

## Config Tab

The `Config` tab controls the menu keybind.

### Menu Bind

Default menu key:

```text
K
```

How to change it:

```text
Right-click the menu bind button
Press the new key
The menu key updates
```

### Menu Bind Button

```text
Left-click menu bind button = open/close menu
Right-click menu bind button = rebind menu key
```

---

## Keybind System

Most toggles can be rebound.

How to bind a feature:

```text
Right-click the feature button
Press the key you want
The button updates with the new bind
```

Example:

```text
Fly: OFF | Bind: J
```

After rebinding, pressing that key toggles the feature.

---

## Mouse and Camera Behavior

When the menu is open:

```text
Your mouse cursor is shown
The menu can be clicked
The camera mode is not changed
MouseBehavior is not forced
```

When the menu is closed:

```text
The mouse icon goes back to how it was before
The camera should stay normal
```

---

## Quick Controls Summary

```text
K = open/close menu
H = toggle noclip
J = toggle fly
WASD = fly movement
Space = fly up
LeftControl = fly down
Left-click toggle = turn feature on/off
Right-click toggle = bind a key
Left-click object group = dropdown
Right-click object group = toggle whole group
Unload = remove the menu and turn off features
```
